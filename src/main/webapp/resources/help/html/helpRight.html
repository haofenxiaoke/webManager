帮助中心
1. 直播过程中听不到声音？

（1）请检查你的电脑设备是否有问题，声卡驱动是否安装正常

（2）请检查你的电脑是否成功连接网络。

2. 无法正常观看视频？

（1）确认你使用了支持视频播放器的浏览器

（2）清除浏览器的浏览数据及缓存

（3）关闭硬件加速。请在浏览器的 设置-显示高级设置中，将“使用硬件加速模式”（如果可用）一项取消勾选，并重新启动浏览器。

（4）检查你的网速。你可以尝试在其它视频网站播放视频（如优酷，土豆等），如果你在其它网站仍无法观看视频，那么你的网络连接很可能有问题。你需要重新连接你的网络或者切换至网速更快的网络以实现流畅的视频播放。

3. 会员有哪些权限？

（1）可以收看网站所有课程

（2）可以阅读网站所有的随堂资料

（3）可以收看所有的直播课程。

4. 学习计划是什么？有什么用途？

（1）学习计划是我们的授课老师，根据你所在的行业和岗位，为你量身打造的学习方案。

（2）按照老师制定的学习计划来学习，可以确保当前的课程是最适合你的。整个学习过程中，将变得更加高效。

（3）在你的个人中心，你能查看到每天的学习进度，以及成长曲线图。让你拥有持久的学习动力。
